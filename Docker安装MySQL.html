Docker 安装 MySQL

本文最后更新于 2023年12月3日 晚上

  1. 下载 MySQL 镜像
  2. 准备映射目录
    群晖目录下自己选择个地方创建这三个目录，分别用于存储配置文件、数据、错误日志。在创建容器时会用到它们并给对应的映射关系
  3. 创建容器
  • 双击镜像列表 mysql 开始创建
  • 在变量中添加“MYSQL_ROOT_PASSWORD”变量，值为密码
  • 添加端口映射
  • 添加文件夹映射
  1. 启动容器
